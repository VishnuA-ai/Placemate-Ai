## 🚀 Quick Start

### Prerequisites
- **Node.js** 16+ ([download](https://nodejs.org))
- **npm** or **pnpm**

### Installation

```bash
# Clone the repository
git clone https://github.com/VishnuA-ai/Placemate-Ai.git
cd "Placement Assistance Application"

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will open at **`http://localhost:5173`** 🎉

### Build for Production

```bash
npm run build
```
